# CAM_HP-NMR
Source code for MRI data processing for the paper "Hyperpolarized 13C Magnetic Resonance Spectroscopy for Chorioallantoic Membrane Assay Studies". 

python_version_checker: check python version installed on computer \\
FunctionsProcessMulti_Longit13C: all functions necessary to process 13C MRI data from a Bruker Maxwell Biospec instrument. \\
ProcessDatPhaseGen_Longit: jupyter notebook set to manually process 13C MRI data from a Bruker Maxwell Biospec instrument. Requires user interaction. \\
